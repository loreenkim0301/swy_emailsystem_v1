## 🔧 버전 전환 가이드

### 실제 배포 버전으로 되돌리기
```bash

cp design-system/v2.1.5/styles.css ./styles.css
```

### 다른 버전으로 전환
```bash
# v2.0.0으로 되돌리기
cp design-system/v2.0.0/styles.css ./styles.css

# v2.2.0으로 업그레이드
cp design-system/v2.2.0/styles.css ./styles.css
```

### 새 버전 생성
```bash
# 현재 스타일을 새 버전으로 저장
mkdir design-system/v2.3.0
cp current-styles.css design-system/v2.3.0/styles.css
```

## 🚀 실제 배포 버전 (v2.1.5) 특징

### **뉴스레터 구독 섹션**
```html
<section class="newsletter-section">
    <h2>📬 바이브코드제로 클럽 구독</h2>
    <form class="newsletter-form">
        <input type="email" placeholder="이메일 주소를 입력하세요">
        <button type="submit">구독하기</button>
    </form>
</section>
```

### **다크 테마 푸터**
```html
<footer class="footer">
    <div class="footer-content">
        <div class="footer-section">바이브코드제로</div>
        <div class="footer-section">학습 도구</div>
        <div class="footer-section">연결하기</div>
    </div>
    <div class="footer-bottom">
        <div class="footer-social">소셜 링크</div>
        <p>저작권 정보</p>
    </div>
</footer>
```

### **CSS 변수 시스템**
```css
:root {
    /* 기존 라이트 테마 */
    --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    --background-card: #ffffff;
    
    /* 새로운 다크 테마 */
    --dark-bg: #1a202c;
    --dark-text: #e2e8f0;
    --dark-border: #4a5568;
}
```
