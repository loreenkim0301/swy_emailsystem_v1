단계별 가이드로 EmailJS를 배우고 실제 이메일 발송까지 테스트해보세요

1

EmailJS 설정

EmailJS 계정 정보를 입력하여 서비스를 초기화합니다.

EmailJS Dashboard → Account → General에서 확인
EmailJS Dashboard → Email Services에서 확인
EmailJS Dashboard → Email Templates에서 확인
테스트 이메일이 발송될 주소입니다
2

연결 테스트

설정한 EmailJS 연결 상태를 확인하고 테스트 이메일을 발송합니다.

3

이메일 발송

실제 이메일을 작성하고 발송해보세요.
